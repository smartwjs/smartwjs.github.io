使用说明

  1. 在GitHub上创建一个新的仓库（或使用现有仓库）
  2. 在 GitHub设置 → Developer settings → Personal access tokens 创建一个新的令牌，需要勾选 repo 权限
  3. 将令牌、用户名和仓库名称填入上方配置区域并保存
  4. 选择文件并上传到您的GitHub仓库
  5. 上传的文件将出现在您的仓库中，可以通过 https://[用户名].github.io/[仓库名]/[文件路径] 访问
  6. 确保您的仓库已启用GitHub Pages：仓库设置 → Pages → Source选择分支（通常是main）
