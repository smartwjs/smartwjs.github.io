# 安装篡改猴（Tampermonkey）教程
详细记录篡改猴的安装步骤，适用于 Edge 和其他浏览器。

## 步骤 1：获取扩展下载地址
- [Edge 官方扩展商店](https://microsoftedge.microsoft.com/addons/Microsoft-Edge-Extensions-Home?hl=zh-CN)
- [备用下载地址](https://www.crxsoso.com/) // 防止官方地址限制

## 步骤 2：搜索“篡改猴”
在扩展商店搜索框输入「篡改猴」，找到**红色图标**的 Tampermonkey 扩展。

## 步骤 3：安装扩展
### Edge 浏览器
点击「安装/获取」按钮 → 在弹出的小窗口中再次点击「安装」，等待安装完成。

### 其他浏览器（Chrome/360 等）
1. 点击黄色下载按钮，选择 `.crx` 后缀的文件下载；
2. 打开浏览器扩展管理界面（通常在「设置→扩展程序」）；
3. 开启「开发者模式」；
4. 将下载的 `.crx` 文件直接拖入扩展界面，确认安装。

## 步骤 4：配置篡改猴权限
打开篡改猴的设置界面，开启以下权限：
- 允许访问 URL
- 允许使用用户脚本
- 允许使用 UserScript

## 步骤 5：安装指定脚本
1. 点击链接：[GreasyFork 脚本地址](https://home.greasyfork.org.cn/zh-hans/info/#/zh-CN/scripts/471106/detail)；
2. 点击页面上的「安装」按钮；
3. 等待跳转到黑色背景的「Script Install」界面；
4. 稍等片刻后，在绿色背景的确认界面再次点击「安装」。

## 步骤 6：验证安装结果
访问 [AtCoder 官网](https://atcoder.jp)，如果出现更新页面，代表所有步骤完成！

> 备注：如果安装失败，可检查浏览器开发者模式是否开启，或换备用扩展下载地址重试。
